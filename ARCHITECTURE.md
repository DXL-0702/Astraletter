## 🔄 数据流

### 星图生成流程

```
用户导入聊天记录
       ↓
   [Parser]
       ↓
   消息数组
       ↓
   [Web Worker]
       ↓
   ┌──────────────────┐
   │ 情感分析 + 嵌入  │
   │ 话题聚类         │
   └──────────────────┘
       ↓
   星图结构数据
       ↓
   [Universe Generator]
       ↓
   3D 位置/颜色/大小
       ↓
   [R3F Render]
       ↓
   显示星图
```

### 数据持久化流程

```
用户保存星图
       ↓
   [AES 加密]
       ↓
   [IndexedDB 存储] (本地)
       ↓
   (可选) [同步到云端]
       ↓
   [加密存储到 Neon]
```

## 🔒 隐私与安全原则

### 核心原则

1. **原始聊天数据永不离开浏览器**
2. **仅存储加密后的星图结构**
3. **AI 计算 100% 本地完成**
4. **云端仅存储匿名统计**

## 📊 性能优化策略

### 3D 渲染优化
- 使用 `InstancedMesh` 渲染所有星星
- `WebGPURenderer` 优先，自动降级
- `MeshStandardMaterial` 移动端优化

### AI 推理优化
- q4/q8 量化模型
- WebGPU 加速
- Web Worker 隔离
- IndexedDB 模型缓存
