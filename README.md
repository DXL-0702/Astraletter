## 🌌 项目简介

**Astraletter** 是一个将私密对话转化为可交互 3D 关系星宇的创新项目。

### 💫 核心价值

把你们的聊天记录，升华为情感艺术品，让你以全新视角回溯、探索、纪念一段关系。

### 🌟 核心亮点

| 特性 | 描述 |
|------|------|
| 🔒 **隐私优先** | 所有数据本地处理，原始聊天记录永不离设备 |
| 🤖 **本地 AI** | Transformers.js 驱动情感分析与话题聚类 |
| 🌌 **3D 星图** | React Three Fiber 打造沉浸式宇宙体验 |
| ⚡ **极致性能** | WebGPU 加速，移动端流畅 |

---

## 🚀 核心功能

### 1. 数据导入与隐私保护

- ✅ 支持微信/WhatsApp TXT 导入
- ✅ 纯文本对话粘贴
- ✅ 本地 OCR 截图提取
- ✅ 端到端加密存储

### 2. 宇宙生成引擎

- ✅ 情感极性分析
- ✅ 智能话题聚类
- ✅ 诗意星座命名
- ✅ 动态星图结构生成

### 3. 3D 交互体验

- ✅ 双视角漫游（宏观/飞行）
- ✅ 点击查看对话气泡
- ✅ AI 星语翻译
- ✅ AR 模式（增强版）

### 4. 长期留存

- ✅ 星空日记
- ✅ 星系博物馆
- ✅ 纪念日提醒

## 🛠️ 技术栈

### 前端
| 层级 | 技术 |
|------|------|
| 框架 | Next.js 14+ |
| 3D 渲染 | React Three Fiber + Three.js |
| 本地 AI | Transformers.js |
| 渲染器 | WebGPU (优先) / WebGL 2 |

### 后端
| 层级 | 技术 |
|------|------|
| 框架 | Hono 4+ |
| 运行时 | Bun 1.3+ |
| 数据库 | PostgreSQL (Neon) |
| ORM | Drizzle ORM |
| 实时通信 | Socket.io |
| 认证 | Lucia Auth |
